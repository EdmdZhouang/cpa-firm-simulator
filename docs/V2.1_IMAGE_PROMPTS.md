## 使用说明

1. 将上述 prompt 直接复制到 AI 图片生成工具（Midjourney / DALL-E / Stable Diffusion / ComfyUI）
2. 建议先生成 Batch 1（tileset），验证 seamless 和透视统一
3. Batch 2 和 Batch 3 可并行生成
4. Batch 4 可在 Batch 2 验收后生成
5. 所有输出必须检查：透明背景、无白边、尺寸正确
6. 命名按 `V2.1_ART_DIRECTION.md` 第 6 节规范执行
